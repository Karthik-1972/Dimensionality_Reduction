# 📊 Dimensionality Reduction Techniques

## 🚀 Overview

This repository demonstrates key **Dimensionality Reduction techniques** used in Machine Learning to reduce feature space while preserving important information.

It includes practical implementations using real datasets and visualization through Jupyter notebooks.

---

## 🧠 Techniques Covered

* Principal Component Analysis (PCA)
* Linear Discriminant Analysis (LDA)

---

## 📁 Project Structure

* `Dataset/` → Contains dataset used for analysis
* `kernel_pca.ipynb` → Kernel PCA implementation
* `principal_component_analysis.ipynb` → PCA implementation
* `linear_discriminant_analysis.ipynb` → LDA implementation

---

## 📌 Problem Statement

High-dimensional data can lead to increased complexity and overfitting.
This project focuses on reducing dimensions while retaining maximum variance and improving model performance.

---
